# Shop Puzzle Simulation

在庫配置パズル型の店舗経営シミュレーションゲームです。

**Play:** https://primevalsistail.github.io/shop-puzzle-simulation/

## ゲーム概要

小さな店のオーナーとして、限られたフロアに形の異なる商品をパズルのように配置しながら売上を伸ばしていくゲームです。

- **フロアパズル** — テトリス状の商品を6×5グリッドに配置。隣接ボーナスを意識した配置が重要
- **クラフト** — 素材を組み合わせて商品を制作。クラフト中は時間が停止
- **経営シミュレーション** — 「進める」ボタンで時間を進め、顧客を呼び込んで売上を積み上げる
- **目標** — 累計売上100万円でエンディング。クリア後はエンドレスモード解放

## 操作方法

| 操作 | 内容 |
|------|------|
| ドラッグ&ドロップ | フロアへ商品を配置 |
| 右クリック / Rキー | 商品を90°回転 |
| 「進める」ボタン | 時間進行（顧客来店・販売） |
| クラフトメニュー | 素材から商品を制作 |
| 仕入れメニュー | 素材・商品を購入 |
