# ユニット・ストーリーマップ

ユーザーストーリーはスキップされたため、代わりに**ゲームプレイシナリオ**をユニットにマッピングします。

---

## Unit 1: Foundation

| シナリオ | 説明 |
|---------|------|
| ゲーム起動 | ブラウザでゲームが起動し、ゲーム画面が表示される |
| 時間進行 | 「進める」ボタンを押すと時間が高速で進む |
| 時間停止 | クラフト操作中は時間が停止する |

---

## Unit 2: Grid & Placement

| シナリオ | 説明 |
|---------|------|
| グリッド表示 | 店舗フロアのグリッドが画面に描画される |
| アイテム配置 | 倉庫からアイテムをドラッグしてグリッドに置ける |
| アイテム回転 | 配置前にアイテムを90°単位で回転できる |
| 配置制約 | 範囲外・他アイテムと重なる場所には置けない |
| 隣接ボーナス表示 | 相性のあるアイテムが隣接するとハイライト表示される |
| 陳列スタック | 同じアイテムを同じ陳列スペースに最大999個まで積める |

---

## Unit 3: Items, Crafting & Inventory

| シナリオ | 説明 |
|---------|------|
| 仕入れ | 仕入れメニューから素材・商品を購入して倉庫に入れる |
| 倉庫上限（種類） | 種類数の上限に達すると新しい種類を仕入れられない |
| 倉庫上限（数量） | 1種類あたり999個を超えて仕入れできない |
| クラフト実行 | クラフトメニューでレシピを選んで素材→製品を作る |
| クラフト時間停止 | クラフト中は「進める」が無効になり時間が止まる |
| レシピアンロック | ゲーム進行に応じて新しいクラフトレシピが解放される |
| 倉庫→フロア移動 | 倉庫のアイテムをフロアの陳列スペースに移す |

---

## Unit 4: Sales & Economy

| シナリオ | 説明 |
|---------|------|
| 顧客来店 | 「進める」を押すと顧客がランダムに来店する |
| 商品購入 | 顧客が棚の商品を購入し、数量が減る |
| 隣接ボーナス効果 | 隣接ボーナスがある商品は購買率・売上が上昇する |
| 売上計上 | 売れた商品の売上が資金に加算される |
| 売り切れ通知 | 陳列数量が0になると「補充 / 撤去」プロンプトが表示される |
| 補充 | 倉庫から在庫を補充して陳列を継続する |
| 撤去 | 陳列スペースを撤去してグリッドを解放する |
| 仕入れコスト | 仕入れで資金が減る。残高不足時は仕入れ不可 |

---

## Unit 5: UI & Progression

| シナリオ | 説明 |
|---------|------|
| HUD表示 | 画面上部に資金・累計売上・目標残りが常時表示される |
| 目標達成 | 累計売上100万円でエンディング画面が表示される |
| エンドレス移行 | エンディング後にエンドレスモードが解放される |
| ゲームオーバー | 資金がゼロになるとゲームオーバー画面が表示される |
| グリッド拡張 | 条件達成でフロアが拡張される（6×5→9×7→13×10） |
| セーブ | ゲーム状態がLocalStorageに自動または手動で保存される |
| ロード | 前回のセーブデータから再開できる |
| チュートリアル | 初回プレイ時に基本操作のガイドが表示される |
